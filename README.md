# Task Management Application

A simple task management application to create, track, and manage tasks with priorities and deadlines.

## Features:
- Create tasks with titles, descriptions, priorities, and deadlines.
- View tasks in list format or calendar view.
- Sort tasks by priority or due date.
- Mark tasks as "To Do", "In Progress", or "Completed".
- Get notifications for upcoming deadlines.
